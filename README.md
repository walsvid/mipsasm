# mipsasm
![](https://img.shields.io/badge/qt-5.10.1-brightgreen.svg) ![](https://img.shields.io/badge/macOS-10.13.4-blue.svg)

## 简介
一个简单的 MIPS 汇编器，能够将汇编语句转换成二进制的01字符串
目前完成了对基本语句的转换

## 功能

- 实现从汇编代码到机器码的汇编过程
- 存储读取文件
- 复制粘贴等基本文本操作
- GUI界面
- 双栏操作
- 代码高亮
